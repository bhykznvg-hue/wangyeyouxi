# 骑砍风云（Mount & Blade Web）

一款纯网页版的开放世界「骑马与砍杀」风格 3D 游戏。基于 **Three.js (WebGL2)** 开发，
所有模型、贴图、音效均为**程序化生成的原创内容**，不包含任何《骑马与砍杀》原作素材。

## 运行方式

- 方式一：直接双击 `index.html`（已内置本地 three.min.js，无需联网）。
- 方式二（推荐）：在本目录启动任意静态服务器，例如：
  - `python -m http.server 8080` 然后访问 `http://localhost:8080`
- 建议使用最新版 Chrome / Edge，独立显卡可获得最佳体验。

## 核心特性

- **开放世界**：4km × 4km 无缝大地图，平原 / 森林 / 山地 / 雪峰 / 湖泊，程序化地形与道路
- **聚落**：3 座城镇、3 座城堡、6 个村庄，围墙、城门、箭塔、市场、民居均为程序化建筑
- **昼夜循环 + 天气系统**：晴 / 多云 / 雨 / 暴风雨（含闪电）/ 大雾，动态太阳月亮、星空、火把照明
- **第三人称战斗**：方向性劈砍（左挥/右挥/上劈/突刺）、格挡、盾牌、翻滚闪避、弓箭抛物线弹道
- **骑战**：自由骑乘、冲撞践踏、骑枪夹持冲锋（速度加成伤害）、马匹独立血量与死亡
- **战术系统**：士气与溃逃、盾墙挡箭、长枪驻守克骑兵、骑兵迂回冲锋、远程支援、将领 AI（列阵→推进→侧翼包抄→总攻）
- **指挥系统**：按兵种分组（步/弓/骑），跟随 / 驻守 / 冲锋指令，横列 / 盾墙 / 楔形 / 散阵 / 环形五种阵型
- **军队养成**：招募 → 经验升级兵种树（新兵→步兵/弓手/骑兵三线，最高至卫士/神射手/骑士）、军饷系统
- **RPG 系统**：等级 / 技能（近战、箭术、骑术、统御、交易）、装备（武器/护甲/盾/战马）、背包
- **经济**：城镇差价贸易、商店、战利品、商队
- **阵营与声望**：三大王国 + 强盗，动态宣战/停战，声望决定敌对关系
- **任务**：剿匪、运货、讨伐强盗团伙
- **野战与攻城**：世界漫游部队实时遭遇；攻城战需破坏城门、清剿城墙弓手与守军，可占领城堡
- **性能优化**：全兵种实例化渲染（人/马部件级 InstancedMesh）、地形 LOD、动画距离降频、空间哈希碰撞，支持 200~300+ 单位同屏混战
- **存档**：暂停菜单中保存 / 读取（localStorage）

## 新手路线

1. 出生在城镇门口，先进城找**募兵官/长老**招募几名新兵（村庄也可招募）。
2. 在附近平原剿灭小股**强盗**（地图 M 上的深色小点），拾取战利品箱。
3. 用金币购买**皮甲、精钢剑、猎弓、战马**，升级士兵为步兵/弓手/骑兵。
4. 接**村庄/领主任务**赚钱刷声望；注意每天要发军饷。
5. 军队成型后（建议 25+ 高级兵），对敌对阵营城堡按 E 发动**攻城战**：
   选中部队按 C 冲锋 → 士兵自动砍破城门 → 歼灭守军即可占领。

## 战术提示

- 敌方箭雨来袭时：选中步兵按 V 切到**盾墙**并 G 驻守，正面挡箭。
- 敌方骑兵冲锋时：**长枪兵**驻守会自动架枪拒马，战马撞上会人仰马翻。
- 骑枪夹持（X）在高速下命中可造成巨额伤害；冲撞可击倒敌方步兵。
- 士气过低的部队会溃逃：集中击杀敌军、保持人数优势可加速敌方崩溃。

## 技术说明

- Three.js r147（本地 `lib/three.min.js`），WebGL2 + PBR 材质（MeshStandardMaterial）+ ACES 色调映射 + PCF 软阴影
- 士兵/马匹为部件级 InstancedMesh 池（17+8 个绘制调用渲染全部单位），程序化 FK 骨骼动画
- 地形：种子化 FBM 噪声高度场 + 双线性采样，20×20 分块三级 LOD，顶点色生物群系
- `test-node.js`：Node 环境无头模拟测试（世界生成 / 遭遇战 / 攻城全流程）：`node test-node.js`
